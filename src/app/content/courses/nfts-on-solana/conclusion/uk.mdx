# Висновок

Вітаємо! Ви завершили комплексний курс про NFT на Solana та розвинули глибоке розуміння невзаємозамінних токенів у екосистемі Solana.

<ArticleSection name="Що ви вивчили" id="what-you-learned" level="h2" />

Протягом цього курсу ви отримали важливі знання про:
- Основні властивості NFT: неподільність, унікальність, фіксована пропозиція, групованість та метадані
- Програму Metaplex Token Metadata: `Metadata`, `Master Edition` та `Token Record` акаунти
- Програму Metaplex Core: дизайн з одним акаунтом для підвищення ефективності та їхню систему плагінів
