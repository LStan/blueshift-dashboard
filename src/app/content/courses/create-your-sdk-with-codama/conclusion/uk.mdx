# Висновок

Вітаємо! Ви завершили курс `Create your SDK with Codama`. Тепер у вас є міцна основа для розуміння того, як працює Codama IDL, від основних концепцій до практичних деталей реалізації.

## Що ви вивчили

Протягом цього курсу ви отримали важливі знання про:
- Що таке Codama і як використовуються `nodes`, `visitors` та `renderers`
- Як створити Codama IDL з нуля
- Як створити Codama IDL з Anchor IDL та оновити його за потреби

## Наступні кроки

Тепер ви готові почати створювати свій перший SDK з Codama! Найкращий спосіб закріпити свої знання — це практика. Ми рекомендуємо вам:

1. Перейти до [розділу Завдання](/uk/challenges)
2. Створити SDK для однієї з програм
3. Використати SDK для створення тестів для вашої програми
4. Приєднатися до нашої спільноти, щоб поділитися своїм прогресом і отримати допомогу
