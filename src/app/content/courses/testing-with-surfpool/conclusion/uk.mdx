# Висновок

Вітаємо! Ви завершили курс тестування з Surfpool. Тепер у вас є комплексне розуміння того, як використовувати Surfpool — потужний фреймворк для тестування, який переносить стан мережі mainnet безпосередньо у ваше локальне середовище розробки для безперешкодного тестування міжпрограмних викликів (CPI).

<ArticleSection name="Що ви вивчили" id="what-you-learned" level="h2" />

Протягом цього курсу ви опанували основні концепції Surfpool:
- **Симуляція мережі mainnet**: Як Surfpool отримує акаунти з мережі mainnet у режимі реального часу, усуваючи складне налаштування для тестування CPI
- **Розширене керування**: Подорожі в часі до будь-якої епохи/слоту/часової мітки, призупинення створення блоків та маніпулювання системними змінними
- **Модифікація акаунтів**: Налаштування параметрів постачання SOL, передача акаунтів між програмами та оновлення даних токен-акаунтів

<ArticleSection name="Наступні кроки" id="next-steps" level="h2" />
